# BG3 Spell Slot Macro

 This is an AutoHotkey script that automatically creates infinite spell slots using the Potion of Angelic Reprieve
method. 

# Requirements

- You need AutoHotkey2 installed
- BG3 needs to be in Borderless Window
- BG3's UI scale must be set to default

# How to Use

 Place your Create Sorcery Point spell in a way that when clicking it your pointer is automatically on
 top of the level 2 option without needing to move the mouse. Then place Create Spell Slot to the left
 of it and your potions of Angelic Reprieve to the right. Press F9 and introduce the right inputs.

![Demo](assets/bg3ssdemo.gif)

# Hotkeys

- F9: Starts the full script
- F10: Manually starts the "Create Spell Slot/Sorcery Point" loop n times
- F12: Reloads the script
